# Vanilla JS SPA with Routing

**Complexity:** Intermediate
**Stack:** Vanilla JavaScript + Silcrow (No Framework!)

## Use Case
Build a complete single-page application with client-side routing using only vanilla JavaScript and Silcrow. No frameworks, no build tools.

## What This Demonstrates
- Simple client-side router (< 50 lines)
- Multiple pages with Silcrow data binding
- Browser history API integration
- Zero dependencies except Silcrow

## File Structure
```
03-vanilla-spa/
├── index.html        # Complete SPA in one file
└── README.md
```

## Run
```bash
# Serve with any static server
python -m http.server 8000
# or
npx serve .
# Open http://localhost:8000
```

## Key Learnings
- You don't always need React/Vue/Angular
- Vanilla JS + Silcrow is powerful for many use cases
- Client-side routing is simple with History API
- Progressive enhancement friendly
